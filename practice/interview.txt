* 제이쿼리(jQuery)
웹 사이트에 자바스크립트를 쉽게 활용할 수 있도록 도와주는
오픈소스 기반의 자바스크립트 라이브러리

웹 페이지 상의 엘리먼트를 쉽게 찾고, 조작할 수 있다.
모든 웹 브라우저에 대응할 수 있다. (호환성 뛰어남)
다양한 기능, 유지관리가 용이한 코드 작성 지원
플러그인 구현/확장 가능

* WEB & WAS
WEB Server : 웹 브라우저 클라이언트가 요청을 보내면
그 명령에 대한 처리를 실행하고
다시 사용자에게 답변을 보내주는 역할을 수행

WAS Server : Web Application Server
웹 서버의 부하를 분산하기 위해
웹서버(정적 data 처리) + 웹 컨테이너(동적 data 처리)로 웹 상에서 사용하는 컴포넌트를 올려놓고 사용하는 서버
서버단에서 어플리케이션을 동작할 수 있도록 지원

* 오버라이딩 & 오버로딩
오버라이딩 : 상속 받은 클래스의 메소드를 재정의
오버로딩 : 동일한 메소드명을 가진 메소드를 매개변수의 타입이나 갯수를 다르게 하여 구현

* Java Collection
List : 순서 O, 데이터 중복 O
Set : 순서 X, 데이터 중복 X
Map : Key & Value 저장, Key 중복 X, Value 중복 O
